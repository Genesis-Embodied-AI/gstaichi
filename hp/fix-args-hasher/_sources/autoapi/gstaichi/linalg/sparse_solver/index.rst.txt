.. py:class:: SparseSolver(dtype=f32, solver_type='LLT', ordering='AMD')

   Sparse linear system solver

   Use this class to solve linear systems represented by sparse matrices.

   :param solver_type: The factorization type.
   :type solver_type: str
   :param ordering: The method for matrices re-ordering.
   :type ordering: str


   .. py:attribute:: matrix
      :value: None



   .. py:attribute:: dtype


   .. py:method:: compute(sparse_matrix)

      This method is equivalent to calling both `analyze_pattern` and then `factorize`.

      :param sparse_matrix: The sparse matrix to be computed.
      :type sparse_matrix: SparseMatrix



   .. py:method:: analyze_pattern(sparse_matrix)

      Reorder the nonzero elements of the matrix, such that the factorization step creates less fill-in.

      :param sparse_matrix: The sparse matrix to be analyzed.
      :type sparse_matrix: SparseMatrix



   .. py:method:: factorize(sparse_matrix)

      Do the factorization step

      :param sparse_matrix: The sparse matrix to be factorized.
      :type sparse_matrix: SparseMatrix



   .. py:method:: solve(b)

      Computes the solution of the linear systems.
      :param b: The right-hand side of the linear systems.
      :type b: numpy.array or Field

      :returns: The solution of linear systems.
      :rtype: numpy.array



   .. py:method:: info()

      Check if the linear systems are solved successfully.

      :returns: True if the solving process succeeded, False otherwise.
      :rtype: bool
