.. py:class:: PLYWriter(num_vertices: int, num_faces=0, face_type='tri', comment='created by PLYWriter')

   Writes `numpy.array` data to `ply` files.

   :param num_vertices: number of vertices.
   :type num_vertices: int
   :param num_faces: number of faces.
   :type num_faces: int, optional
   :param face_type: `tri` or `quad`.
   :type face_type: str
   :param comment: comment message.
   :type comment: str

      Sets the (r, g, b) channels of the colors at the vertices.
