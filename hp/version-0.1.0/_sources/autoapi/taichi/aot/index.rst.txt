taichi.aot
==========

.. py:module:: taichi.aot

.. autoapi-nested-parse::

   Taichi's AOT (ahead of time) module.

   Users can use Taichi as a GPU compute shader/kernel compiler by compiling their
   Taichi kernels into an AOT module.
