# Supported systems

## CI Tested systems

We test the following systems in our CI servers:
- python 3.10-python 3.11
- mac os x 14 and 15
- Ubuntu 22.04 and Ubuntu 24.04
- Windows Server 2025

## Supported systems:

### Operating systems
- Mac silicon 14 or 15
- Ubuntu 22.04 and 24.04 x86
- Windows 10 or later

### GPUs

- CUDA GPUs, Pascal or later (>=sm_60)
- Metal GPUs
- Vulkan-compatible GPUs
